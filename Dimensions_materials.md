## **1\. Repainting / Refinishing**

**What you're measuring:** Every painted surface. You count each component separately because they have different dimensions and painting times.

**Component counts you need:**

| Component | What to count | Why it matters |
| :---- | :---- | :---- |
| Upper cabinets | Number of boxes | Determines number of doors \+ face frame area |
| Upper doors | Count each door leaf | Some uppers have 1 door, some have 2 |
| Lower cabinets | Number of boxes | Determines door \+ drawer front count |
| Lower doors | Count each door leaf |  |
| Tall / pantry cabinets | Number of boxes | Taller surface \= more paint sq ft |
| Tall doors | Count each door leaf (usually 2 per tall) |  |
| Drawers | Count actual drawer boxes | Affects drawer front face area |
| Drawer fronts | Count each visible front panel | Some cabinets have 2–4 stacked drawer fronts |
| Dummy / false drawers | Count each | These are fixed panels with no box behind — painted same as real fronts |
| Cabinet crown molding | Linear feet | Painted separately, different technique (brush vs spray) |
| Face frame stiles | Count per cabinet, measure width | Vertical frame members — cut-in painting |
| Face frame rails | Count per cabinet, measure width | Horizontal frame members — top, middle, bottom |
| Toe kick | Linear feet | Painted or left raw |
| Cabinet box interior | Count boxes being painted inside | Optional but common on full refinish jobs |

**Dimensions needed per component:**

* **Each door** — width × height (both faces if painting back)  
* **Each drawer front** — width × height  
* **Each dummy drawer front** — width × height (same as real front visually)  
* **Each face frame stile** — width × height (per cabinet box)  
* **Each face frame rail** — width × length  
* **Crown molding** — profile width × linear feet  
* **Toe kick** — height × linear feet  
* **Box interior** — interior width × height × depth (if painting inside)

**Required Materials:**

* Liquid deglosser or TSP cleaner  
* Chemical stripper (if heavy existing finish)  
* 120, 150, 180 grit sandpaper  
* Grain filler / wood conditioner (for open-grain wood like oak)  
* Tack cloth  
* Bonding primer (1–2 coats)  
* Paint or stain — gallons calculated from total sq ft (1 gallon covers \~350 sq ft)  
* Clear topcoat / sealer (polyurethane, lacquer, or conversion varnish)  
* Painter's tape & plastic sheeting  
* Brush, roller, or spray equipment  
* Touch-up brush (for frame cut-in work)  
* Drop cloths

## **2\. Refacing — New Doors \+ Veneer**

**What you're measuring:** Every new part being ordered (doors, drawer fronts, dummy panels), every hinge position, and every face frame surface being veneered.

**Component counts you need:**

| Component | What to count | Why it matters |
| :---- | :---- | :---- |
| Upper cabinets | Number of boxes | Determines veneer sq ft \+ hinge plates |
| Upper doors | Count each door leaf | Ordered individually to exact size |
| Lower cabinets | Number of boxes |  |
| Lower doors | Count each door leaf |  |
| Tall / pantry cabinets | Number of boxes |  |
| Tall doors | Count each door leaf (usually 2 — upper \+ lower) | Height determines hinge count (2, 3, or 4 per door) |
| Drawers | Count actual drawer boxes | Determines drawer slide condition check |
| Drawer fronts | Count each visible front panel | Ordered to exact width × height |
| Dummy / false drawer fronts | Count each | Ordered same as real fronts — no box behind, fixed with clips or screws |
| Cabinet crown molding | Linear feet \+ profile dimensions | Replaced or repainted to match new doors |

**Dimensions needed per component:**

**Upper doors (per door leaf):**

* Door width (opening width \+ overlay, typically \+1" total for standard overlay)  
* Door height  
* Number of hinges per door (door height up to 40" \= 2, 40"–60" \= 3, 60"+ \= 4\)

**Lower doors (per door leaf):**

* Door width  
* Door height

**Tall / pantry doors (per door leaf):**

* Door width  
* Door height (can be 60"–84" — always needs 3–4 hinges)

**Drawer fronts (per front):**

* Drawer front width  
* Drawer front height

**Dummy / false drawer fronts (per panel):**

* Width  
* Height (same ordering process as real drawer fronts)

**Face frame/box face (for veneer):**

* Box external width  
* Box external height  
* Face frame stile width (left and right vertical members)  
* Face frame rail width (top and bottom horizontal members)  
* Any mid-rail width (horizontal divider between a door and drawer)

**Critical hinge dimensions (per door):**

* Face frame stile width → determines overlay type → determines hinge arm length  
* Door thickness (standard 3/4")  
* Hinge cup hole diameter (standard 35mm)  
* Hinge boring depth (standard 13mm)  
* Edge distance from door edge to center of hinge cup (typically 3–5mm)  
* Overlay amount:  
  * Full overlay: door covers full stile → shorter arm  
  * Half overlay: shared stile, each door covers half → medium arm  
  * Inset: door sits flush inside frame → longer arm, exact fit critical

**Shelf dimensions (if replacing shelves):**

* Shelf length (interior box width − 1/8" clearance)  
* Shelf depth (interior box depth − 1/2")  
* Shelf thickness (standard 3/4")

**Crown molding:**

* Linear feet (measure along top front edge of upper cabinets)  
* Profile height and projection (depth from cabinet face)

**Required Materials:**

* New doors (per exact size per opening)  
* New drawer fronts (per exact size)  
* New dummy / false drawer front panels  
* Veneer sheets (peel-and-stick or contact cement type) — sq ft \= total face frame area  
* Veneer edge banding (iron-on) — linear feet of all exposed box edges  
* Contact cement or veneer adhesive  
* J-roller  
* Concealed hinges (2–4 per door × total door count)  
* Hinge mounting base plates / euro clips (1 per hinge)  
* Soft-close dampers (if upgrading)  
* Door handles / knobs (1 per door)  
* Drawer pulls (1 per drawer front \+ dummy fronts)  
* Drawer slides (replace if worn — 1 pair per drawer)  
* Shelf pins (4 per adjustable shelf)  
* Mounting screws (hinge screws, handle screws, drawer front screws)  
* Cam locks & connector bolts (if any box needs reinforcing)  
* Door bumper pads (4 per door)  
* Magnetic catches (1 per upper door if no soft-close)  
* Wood filler & touch-up for box repairs before veneering  
* New crown molding (if replacing, measured in linear feet)

## **3\. Full Replacement**

**What you're measuring:** Everything — the space, every new box, every door, every piece of hardware, countertop, and trim. This is a full kitchen survey.

**Component counts you need:**

| Component | What to count | Why it matters |
| :---- | :---- | :---- |
| Upper cabinets | Number of new boxes needed | Each ordered by width; must fill wall run exactly |
| Upper doors | Count each door leaf per box |  |
| Lower cabinets | Number of new boxes needed |  |
| Lower doors | Count each door leaf per box |  |
| Tall / pantry cabinets | Number and position | Often floor-to-ceiling; affects layout planning |
| Tall doors | Count per box (usually 2 sets — upper \+ lower) |  |
| Drawer base cabinets | Number of boxes | Drawer base is a separate cabinet type from door base |
| Drawer boxes | Count each actual drawer box inside each cabinet | A 3-drawer base has 3 drawer boxes |
| Drawer fronts | Count each visible front panel |  |
| Dummy / false drawer fronts | Count each | Common on sink base (no real drawer behind the sink) and corner cabinets |
| Cabinet crown molding | Linear feet \+ profile | New crown ordered and installed to match new style |
| Toe kick | Linear feet |  |
| Filler strips | Count and measure each gap | Fill spaces between cabinets and walls |
| Valance / light rail | Linear feet (under upper cabinets) | If adding under-cabinet lighting |

**Dimensions needed per component:**

**Kitchen space (survey first):**

* Total wall length — upper run (each wall separately)  
* Total wall length — lower run  
* Ceiling height  
* Height from floor to bottom of upper cabinets (standard 54" from floor, or 18" above counter)  
* Soffit dimensions — height, depth, and distance from ceiling (if present)  
* Window rough opening — width, height, sill height from floor  
* Door rough opening — width, height, distance from corners  
* Electrical outlet positions (determines where cabinets can and cannot go)  
* Plumbing location — sink drain and supply pipe positions

**Per upper cabinet box:**

* Box width (9"–36" in 3" increments)  
* Box height (30", 36", or 42")  
* Box depth (standard 12")  
* Wall thickness (3/4")  
* Number of doors per box (1 or 2\)  
* Number of shelves inside  
* Shelf pin hole spacing (32mm system standard)

**Per lower cabinet box:**

* Box width  
* Box height (standard 34.5" — countertop brings to 36")  
* Box depth (standard 24")  
* Door configuration (1 door, 2 doors, or drawer bank)  
* Number of drawer boxes inside (for drawer base cabinets)  
* Drawer box width \= box interior width − 1" (slide clearance)  
* Drawer box height (3.5"–5" typical)  
* Drawer box depth \= box interior depth − 1"

**Per tall / pantry cabinet box:**

* Box width  
* Box height (84", 90", or 96" — floor to ceiling)  
* Box depth (24")  
* Door split (upper doors \+ lower doors, with fixed or adjustable shelf between)  
* Number of interior shelves

**Drawer fronts (per front):**

* Same as refacing — width × height per front  
* Reveal gap between stacked drawer fronts (standard 1/8")

**Dummy / false drawer fronts:**

* Width × height  
* Mounting method (clips, screws from inside box)  
* Typical locations: sink base, corner base, beside tall cabinets

**Cabinet crown molding:**

* Linear feet (measure each run, include returns at walls)  
* Crown profile — height × projection (e.g., 3.5" × 2.5")  
* Stacking crown (some styles use a light rail below \+ crown above)

**Countertop:**

* Total linear feet of counter run  
* Countertop depth (standard 25.5")  
* Overhang at front (standard 1.5")  
* Sink cutout — exact sink model dimensions (length × width × mounting type)  
* Corner configuration — straight, L-shape, or U-shape (affects miter cuts)

**Required Materials:**

Cabinet boxes:

* 3/4" plywood or MDF (box sides, top, bottom, shelves)  
* 1/4" plywood (back panel)  
* Edge banding (all exposed plywood edges)  
* Pocket screws (box assembly)  
* Cabinet screws / connecting bolts (joining boxes together)  
* Wall mount screws (into studs — typically 3" screws)  
* Wall anchors (where studs aren't available)  
* Shims (leveling — plastic or wood)  
* Filler strips (plywood or solid wood)  
* Toe kick boards (MDF or hardwood)  
* Toe kick clips / brackets

Doors and hardware (same as refacing, plus):

* Drawer box material (plywood or pre-made dovetail boxes)  
* Undermount or side-mount drawer slides (1 pair per drawer)  
* Soft-close hinges (2–4 per door)  
* Soft-close drawer slides  
* All handles, knobs, pulls  
* All screws, cam locks, shelf pins, bumpers, catches  
* Dummy front clips or screws

Crown and trim:

* Crown molding (linear feet \+ 15% waste)  
* Light rail / valance (linear feet)  
* Toe kick (linear feet \+ 10% waste)  
* Painter's caulk (all gaps between cabinets and walls)

Countertop (if included):

* Countertop slab (sq ft \+ 10% waste)  
* Sink clips and silicone sealant  
* Backsplash material  
* Countertop adhesive / supports
